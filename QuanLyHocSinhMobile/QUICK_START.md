## 🔥 TOP 5 CÚ PHÁP PHẢI BIẾT

### 1️⃣ @Entity (Database Table)
**File:** Diem.java
**Ý nghĩa:** Khai báo bảng SQL
```java
@Entity(tableName = "Diem",
        primaryKeys = {"maHS", "maMH", "hocKy"},
        foreignKeys = {...},
        indices = {...})
public class Diem { }
```
**Cần biết:** primaryKeys, foreignKeys, indices, @Ignore

---

### 2️⃣ Repository Pattern
**File:** TaiKhoanRepository.java
**Ý nghĩa:** Lớp trung gian giữa UI và Database
```java
public class TaiKhoanRepository {
    public TaiKhoanRepository(Application app) {
        AppDatabase db = AppDatabase.getDatabase(app);
        taiKhoanDAO = db.taiKhoanDAO();
    }
    
    public List<TaiKhoan> getAllTaiKhoan() {
        return taiKhoanDAO.getAll();
    }
}
```
**Lợi ích:** Dễ test, dễ maintain, thay đổi DB không ảnh hưởng UI

---

### 3️⃣ SQL CROSS JOIN
**File:** KhoiTaoDatabase.java
**Ý nghĩa:** Kết hợp tất cả hàng từ 2 bảng
```java
"SELECT h.MaHS, m.MaMH FROM HocSinh h CROSS JOIN MonHoc m"
// 50 HS × 10 MH = 500 rows
```
**Dùng cho:** Tạo grid data (HS × MH)

---

### 4️⃣ Generic <T>
**File:** ExcelHelper.java
**Ý nghĩa:** Một method dùng cho nhiều type
```java
public static <T> void exportToExcel(..., List<T> data, 
                                      BiConsumer<Row, T> rowMapper) {
    // T có thể là HocSinh, Diem, GiaoVien, etc.
}
```
**Lợi ích:** Tái sử dụng code, không lặp

---

### 5️⃣ Lambda Expression
**File:** ExcelHelper.java
**Ý nghĩa:** Cách viết tắt anonymous function
```java
(row, hs) -> {
    row.createCell(0).setCellValue(hs.getMaHS());
    row.createCell(1).setCellValue(hs.getHoTen());
}
```
**Tương đương:** Anonymous class dài dòng

## ❓ FAQ NHANH

**Q: Repository dùng để làm gì?**
> A: Lớp trung gian giữa UI (Activity) và Database (DAO). Nếu database thay đổi, chỉ cần sửa Repository, không sửa Activity.

**Q: @Entity là gì?**
> A: Đánh dấu class là một bảng trong SQLite database. Room sẽ auto-generate SQL create table.

**Q: @ForeignKey CASCADE là gì?**
> A: Nếu xóa parent → tự động xóa tất cả child. Ví dụ: xóa học sinh → tự động xóa điểm của HS đó.

**Q: CROSS JOIN là gì?**
> A: Kết hợp mỗi hàng từ bảng 1 với TẤT CẢ hàng từ bảng 2. 50 × 10 = 500 rows.

**Q: Generic <T> dùng để làm gì?**
> A: Một method có thể nhận bất kỳ type nào. exportToExcel<HocSinh>, exportToExcel<Diem>, etc.

**Q: Lambda là gì?**
> A: Cách viết ngắn gọn anonymous function. Dùng thay thế vài dòng code bằng 1 dòng.

**Q: Tại sao dùng volatile trong Singleton?**
> A: Đảm bảo tất cả threads thấy giá trị mới nhất của variable. Prevent cache stale value.

**Q: allowMainThreadQueries() có nguy hiểm không?**
> A: ⚠️ CÓ! Nó sẽ freeze UI khi query database. Chỉ dùng khi test, không dùng production.
